# GOLDSANGSA 개발 계획

## 스택
- Frontend: Next.js (App Router, TypeScript, Tailwind)
- Backend: NestJS (REST API, TypeORM)
- Database: PostgreSQL
- 이미지: 서버 로컬 저장 (/uploads)
- 실행 환경: Docker Compose

---

## 진행 순서

### 1단계: Sanity 제거 ✅
### 2단계: NestJS + PostgreSQL 세팅 ✅
### 3단계: NestJS API 구현 ✅

### 5단계: 관리자 페이지 구현 ✅
### 6단계: Docker Compose 세팅 ✅

## 로컬 실행 방법

### PostgreSQL만 Docker로 실행 (개발 권장)
```bash
# PostgreSQL 실행
docker run -d \
  --name goldsangsa-db \
  -e POSTGRES_DB=goldsangsa \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_PASSWORD=password \
  -p 5432:5432 \
  postgres:16-alpine

# 백엔드 실행
cd backend && npm run start:dev

# 프론트엔드 실행
cd frontend && npm run dev
```

### 전체 Docker Compose 실행
```bash
docker-compose up --build
```
